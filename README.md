# 🤖 AI Dev Prompts
> Archivos `.md` optimizados para usar con IA y mejorar tu código, arquitectura y seguridad.

## 🎯 ¿Qué es este repositorio?
Una colección abierta de documentos `.md` diseñados específicamente para ser usados como **contexto, instrucciones o prompts** en modelos de IA (ChatGPT, Claude, Gemini, Copilot, etc.). Cada archivo contiene buenas prácticas, principios de diseño, guías de seguridad y estándares por tecnología, listos para que la IA los aplique directamente a tu código.

## 🤖 ¿Cómo usar estos archivos con IA?
1. Copia el contenido del `.md` que necesites.
2. Pégalo en tu chat de IA como **contexto inicial** o **instrucción de sistema**.
3. Añade tu código o describe tu problema.
4. La IA actuará como un revisor experto basado en ese documento.
